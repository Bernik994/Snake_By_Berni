# Snake_By_Berni
Snake game project in Python 3.14.3 with Pygame-ce 2.5.7

Exe file built with PyInstaller.

When you run the exe, a SmartScreen message may appear indicating that the program is unrecognized.

If you don't want to run exe, you can download the versions of Python and Pygame-ce used to make this program, so you can run the PythonScript file (.py).
